# CosmicMicrowaveBackground_TFG
Trabajo Fin de Grado centrado en la aplicación de modelos de redes neuronales (en concreto CNN y FCN) para la mejora de la resolución del Fondo Cósmico de Microondas. Desarrollo, entrenamiento y comparación de distintos modelos con el objetivo de reconstruir y recuperar señales a partir de datos científicos complejos.
